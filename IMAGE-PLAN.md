# Image Coverage Plan

156 keywords still need illustrations. This document tracks progress across batched PRs, generating ~5 images per batch.

## How to generate

```bash
# Generate specific images (add entries to scripts/generate-illustrations.mjs first)
node scripts/generate-illustrations.mjs --id term-01-bow

# Upload to Cloudinary
node scripts/upload-to-cloudinary.mjs --id term-01-bow
```

After generating, add the term mapping to `src/components/illustrations/paths.ts`.
