## Content

### VARIATION

Charts show North as True (geographic) North. A compass can only point to Magnetic North, which changes with time and according to your position.

The difference between True and Magnetic North is called variation.

If variation is West, magnetic bearing is greater than true bearing.
If variation is East, magnetic bearing is smaller.

for example
With 5°W variation
070°(T) = 075°(M)

With 5°E variation
070°(T) = 065°(M)

Variation for your position is found on the nearest compass rose.

### DEVIATION

Deviation is caused by ferrous metals and electro-magnetic fields on board which will affect the accuracy of the compass.

[Diagram labels — boat interior with sources of deviation:]
- enamel mug near compass
- cockpit speakers
- electronics
- engine
- mobile phone

The ship's compass is swung to check the effect of magnetic influences on board; this will vary as the boat's heading changes.

A card can be produced for your steering compass showing the deviation for each heading.

### How to apply variation and deviation

Chartwork is in °True –
compass courses must be in °Compass

[Circular diagram showing:]
True → (+ W / - E) → Magnetic → (+ W / - E) → Compass

[And reverse arrows showing:]
Compass → (- W / + E) → Magnetic → (- W / + E) → True

### Finding a compass course

True bearing from chart    060°(T)
Variation                  + 7°W
Magnetic bearing           = 067°(M)
Apply deviation from card  - 2°E
Compass course             = 065°(C)

### Checking for deviation

Point the boat straight at a transit and compare results.

[Diagram labels:]
compass 070°(C)
Deviation is 2°W on this heading.
transit 068°(M)
